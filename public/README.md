# Cartella Immagini Socialee

Inserisci in questa cartella tutte le immagini richieste dal sito. Quando le caricherai su GitHub con questi precisi nomi, verranno servite direttamente:

- `logo.png` — Logo principale dell'intestazione e del piè di pagina.
- `spiegazione.png` — Immagine di sfondo nel row "Come Funziona".
- `evento.png` — Immagine di sfondo nel row "Chi Organizza l'Evento".
- `card1.png` — Immagine di sfondo per la Card 1 (Locandine d'impatto).
- `card2.png` — Immagine di sfondo per la Card 2 (Menu e offerte chiare).
- `card3.png` — Immagine di sfondo per la Card 3 (Promo e messaggi pronti).
- `giorgia.png` — Foto di Giorgia Palazzo.
- `giuseppe.png` — Foto di Giuseppe Bonaccorso.
- `manuel.png` — Foto di Manuel Arlotti.
- `noemi.png` — Foto di Noemi Vitone.
- `carlo.png` — Foto di Carlo Baldassini.
